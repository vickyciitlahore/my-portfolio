# Hassan Bokhary — Personal Portfolio (vCard)

A lightweight single-page portfolio for showcasing Hassan Bokhary’s profile, experience, skills, services, and work.

## Features
- **About**: Concise bio and focus areas
- **Resume**: Education and detailed experience
- **Portfolio**: Filterable project gallery
- **Contact**: Map and contact form
- Responsive design, mobile-friendly

## Tech Stack
- HTML5, CSS3, JavaScript
- Ionicons

## Getting Started
1. Open `index.html` directly in your browser, or
2. Serve locally for better routing and caching behavior:
   - Node: `npx serve .` or `npx http-server . -p 8080`
   - Python: `python -m http.server 8080`

Then visit `http://localhost:8080` (or shown URL).

## Customization
- Profile, About, Services, Skills: edit in `index.html` (sidebar and About sections)
- Resume: update Education and Experience timeline in `index.html`
- Portfolio: replace images in `assets/images/` and edit project cards in `index.html`
- Contact: update map embed and contact email/phone in `index.html`
- Avatar: replace `assets/images/my-avatar.png`
